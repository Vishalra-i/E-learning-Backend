API Documentation

User Routes

Register User

POST /api/v1/users/register

Description: Register a new user.

Login User

POST /api/v1/users/login

Description: Login with user credentials.

Logout User

POST /api/v1/users/logout

Description: Logout with user credentials.

Verify User

GET /api/v1/users/:userId/verify/:token

Description: Verify user email using verification token.

Update User

PATCH /api/v1/users/update

Description: Update user details including avatar.

Course Routes

Get All Course

Get /api/v1/courses

Description: Get all available course.

Admin Only

Create Course

POST /api/v1/courses/create

Description: Create a new course.

Admin Only

Delete Course

DELETE /api/v1/courses/:courseId

Description: Delete a course by ID.

Admin Only

Update Course

Patch /api/v1/courses/:courseId

Description: Update a course by ID.

Enrollment Routes

Enroll Course

POST /api/v1/courses/enroll/:courseId

Description: Enroll in a course.

View Enrolled Courses

GET /api/v1/courses/enroll

Description: View all enrolled courses.
